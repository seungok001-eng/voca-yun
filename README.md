# 정철 VOCA — 정철어학원 청당국제캠퍼스 단어 학습 앱

파닉스부터 수능 만점까지, 20단계 단어 학습 + 시험 + 학원 관리자 페이지를 갖춘 단어장 웹앱입니다.
PC·모바일 브라우저에서 모두 동작하며, 모바일에서는 홈 화면에 앱으로 설치(PWA)할 수 있습니다.

## 실행 방법

```bash
cd vocab-app
cp .env.example .env        # 최초 1회
npm install
npx prisma db push          # DB 생성
npx tsx prisma/seed.ts      # 레벨 20개 + 단어 + 데모 계정
npx tsx scripts/topup-words.ts  # (선택) 보충 단어 반영
npm run dev                 # 개발 서버 → http://localhost:3000
```

### 데모 계정 (비밀번호 모두 `1234`)

| 아이디 | 역할 |
|---|---|
| `director` | 원장 (전체 반 관리) |
| `teacher1` | 선생님 (담당 반만) |
| `student1`~`student3` | 학생 (새싹반 A) |

## 주요 기능

### 학생 앱
- **20단계 레벨 지도**: 씨앗(1-4) → 새싹(5-8) → 나무(9-12) → 숲(13-16) → 정상(17-20)
- **5가지 암기 모드**: 카드 암기, 뜻 고르기(4지선다), 스펠링, 받아쓰기(듣기), 매칭 게임
- **미국식 발음 듣기**: 모든 단어 TTS(en-US)
- **시험**: 한→영 타이핑 / 영→한 타이핑 / 혼합 — 선생님 설정에 따름
  - 영→한은 등록된 모든 뜻 + 선생님이 인정한 추가 정답 모두 정답 처리
  - 탈락 시 전체(또는 틀린 것만) **랜덤 순서 재시험**
  - 발음 평가 ON이면 정답 후 마이크로 발음 → 커트라인 미달 시 오답
- **오답노트**: 자동 기록, 오답 단어만 시험 보기, 해결 처리
- **누적 복습(SRS)**: 통과 단어를 3일→7일→21일 주기로 재출제, 일일시험에도 자동 섞기
- **게이미피케이션**: 포인트, 연속 학습 스트릭, 반 랭킹

### 관리자 페이지 (선생님/원장)
- **반·학생 관리**: 계정 생성, 반 배정, 비밀번호 재설정
- **시험 설정 (반 기본값 + 학생별 오버라이드)**
  - 시험 방식(한→영/영→한/혼합), 하루 목표 단어 수, 탈락 기준(오답 허용 수)
  - 재시험 범위(전체/틀린 것만), 품사 엄격 채점 ON/OFF(저학년용 관대 채점)
  - 발음 평가 ON/OFF + 정확도 커트라인(0~100)
- **실시간 모니터링**: 시험 중인 학생 진행률·오답·화면 이탈(부정행위) 감지
- **통계**: 통과율, 정답률, 자주 틀리는 단어 TOP 10, 학생별 성적
- **커스텀 단어장**: 엑셀(xlsx/csv) 업로드 → 반/학생 배정
- **정답 인정**: 학생의 한글 답을 정답으로 추가 승인 → 이후 자동 정답
- **학부모 알림 로그**: 통과/탈락/연속탈락 메시지 자동 생성 (알림톡/SMS 연동 지점)

## 기술 구성

- **Next.js 16** (App Router) + TypeScript + Tailwind CSS 4 — PC/모바일 반응형, PWA
- **Prisma + SQLite** — 운영 전환 시 `schema.prisma`의 datasource만 PostgreSQL로 교체
- **인증**: JWT 세션 쿠키 (원장/선생님/학생 권한 분리)
- **TTS**: Web Speech API(en-US) — 브라우저 내장, 무료
- **발음 평가**: Web Speech 인식 + 유사도 점수(기본, 무료).
  Azure Pronunciation Assessment 연동 시 `POST /api/test/[id]/pron`에 `azureScore`를 전달하면 그대로 사용됨.

## 단어 데이터

`data/words/level-XX.json` (+ `-extra.json`) — 레벨당 60단어 시드가 들어 있습니다.
정식 오픈 전에 레벨당 600단어(총 12,000)로 확장하려면 같은 형식의 JSON을 추가하고
`npx tsx scripts/topup-words.ts`를 실행하면 됩니다. (중복 단어는 자동으로 건너뜀)
